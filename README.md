Dotfiles
======================

My config files for emacs, zsh, tmux, and screen.

Installation
------------

Clone from github.

    cd ~
    git clone https://github.com/hidetai/dotfiles.git dotfiles

Create symbolic links.

    cd ~
    ./dotfiles/install.sh

Vim Setup
---------

Launch vim, and execute `:NeoBundleInstall` to install bundles.

    :NeoBundleInstall


Updating
--------

    cd ~/dotfiles
    git pull
